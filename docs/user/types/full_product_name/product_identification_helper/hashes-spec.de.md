# Hashes - Specification

Liste von Hashes (`hashes`) vom Wertetyp `array` mit mindestens einem Element enthält eine Liste von kryptographischen Hashes, die zur Identifizierung von Dateien verwendet werden können.

``Javascript
"hashes": {
  // ...
  "items": {
    // ...
  }
}
```
